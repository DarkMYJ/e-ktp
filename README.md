### How to Use
1. Clone / Download This Repo
2. Atur Koneksi ke Database kalian pada file .env
   * DB_CONNECTION=mysql
   * DB_HOST=127.0.0.1
   * DB_PORT=3306
   * DB_DATABASE=laravel
   * DB_USERNAME=root
   * DB_PASSWORD=
   
3. Lakukan Migrasi dengan cara mengetik perintah dibawah pada terminal
    ```
     php artisan migrate
    ```
   
4. Run the App
    ```
     php artisan serve
    ```
> Halaman Admin
    ```
     /admin-page
    ```
